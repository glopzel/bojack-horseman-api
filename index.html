BoJack Horseman API

Introduction

The BoJack Horseman API is based on the BoJack Horseman Netflix series. Data for characters in this API include name, species, aliases, first and last appearances, occupation, and other information.

Endpoints

Root URL

The Root endpoint provides the information for all the url endpoints within the API. All Requests are GET requests sent over HTTPS.

Result of searching https://bojack-horseman-api.herokuapp.com/api :

    { 
        "characters": "https://bojack-horseman-api.herokuapp.com/api/characters/{character name}",
        "species": "https://bojack-horseman-api.herokuapp.com/api/characters/species/{character species}",
        "voice": "https://bojack-horseman-api.herokuapp.com/api/characters/voice/{voice person name}",
        "seasons": "https://bojack-horseman-api.herokuapp.com/api/seasons/{number 1 - 6}"
    }

GET Characters

When requesting a character by their name, voice person or species, you will get either an object or an array of objects. Each object will have a minimum of 7 keys, and certain characters will have an additional 1 or 2 keys to indicate their filmography and/or breed.

Details on the keys and values of each character object
Key Value Type Content
name String Name of the character
voice String Voice person
species String The character's species name
aliases Array Character's aliases
first appearance String The first time they appear in an episode
last appearance String The last time they appear in an episode
occupation Array History of occupations
filmography Array Additional key, returns the filmography of the character
breed String Additional key, returns breed of animal

GET Seasons

When requesting a season by the number of it, you will get an object with the season, the year it was released, and the episode names.

Key Value Type Content
season String Number of the season
year String Year of release
episodes Array List of episodes
